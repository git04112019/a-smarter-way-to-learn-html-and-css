1 - HTML & CSS

Notes
    * Hypertext Markup Language (HTML) - document is a text file that tells the browswer how to assemble a webpage.
    * Cascading Stylesheets (CSS) - file that specifies the styling of the page - fonts, colors, colun widths, and the like.

    When naming HTML and CSS files:
    * Use only lower-case characters.
    * Avoid spaces.
    * stick to 0-9, a-z, and_.
